# Sample Images

Place your test images here for testing the Image Description Agent.

## Recommended Test Images

1. **test_image_1.jpg** - A simple scene or object
2. **test_image_2.jpg** - A complex scene with multiple elements
3. **test_image_3.jpg** - An image with text (e.g., sign, poster)

## Supported Formats

- JPG / JPEG
- PNG
- GIF
- BMP
- WEBP

## Where to Get Sample Images

- Use your own photos
- Download from [Unsplash](https://unsplash.com) (free stock photos)
- Download from [Pexels](https://www.pexels.com) (free stock photos)
- Take screenshots

## Testing Tips

1. **Variety**: Test with different types of images
2. **Complexity**: Mix simple and complex scenes
3. **Text**: Include images with visible text
4. **Size**: Keep images under 10MB for faster processing

---

Once you add images here, run:
```bash
python demo.py
```
